## The PR Situation

Forks are welcome to play around with, but heads up: things are moving way too fast right now. Because the codebase is changing constantly, we aren't accepting random PRs at this exact second. 

If you run into actual breaking behavior, open an issue. Keep it strictly technical: give us the input type (DOL, RPX, REL), the exact instruction or function that choked, and the commit hash you used. Vague "it doesn't work" reports or non-technical drama will just get closed. 

## Want your name up there?

We use the `@all-contributors` bot to automate this grid because nobody wants to manually layout markdown tables. 

Once things settle down, you can get yourself on the board by dropping a comment on your PR or issue:

> @all-contributors please add @yourusername for code, tests

The bot will handle it. If you already landed a commit or fixed some weird behavior over on the Discord `#dolrecomp` channel and your avatar is missing, let us know.
